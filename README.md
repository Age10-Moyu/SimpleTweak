# SimpleTweak

_English version, translated by [DeepSeek](https://chat.deepseek.com):_

Oh, hello there!  
This modpack adds a few simple things, such as *a ton* of optimization mods, information displays, mechanic tweaks, and the like, while **absolutely** not altering the core vanilla gameplay. ~~I said so. Hmm. It must be true.~~  
Since the author is from China, you might need to disable the Chinese localization for some mods in the settings and resource pack pages. Sorry for the trouble, but I really don't have the energy to adjust and adapt further, so... that's it =. =  
If you're interested, why not give it a try?

**To play this modpack, you need to install the JDK 25.0.1 provided by AzulZulu, and your device must be a modern Windows system with an AMD64 architecture.  
We have no plans to adapt it for macOS; however, according to feedback, it seemed to work fine on older macOS versions, but we cannot guarantee there won't be unknown and uncontrollable bugs.  
It is not adapted for any Linux distribution, as this modpack and its dependencies cannot run on any known mainstream Linux distributions unless Wine is used.**
